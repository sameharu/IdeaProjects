Create two classes Employee and Address as required below.
a. Address has four fields (data members): street, city, state and zip (all of type
String)
b. Employee has five fields: first name (String), last name (String), salary
(double), home address and work address (both of type Address).
c. Place constructors in each class to initialize all the fields with default
constructor, parameterized constructor(three argument) without address and
parameterized constructor(5 arguments) for Employee class. For Address
class place only one constructor to initialize all the fields.
d. Place appropriate getters and setters in each class.
e. Write a main method which creates two instances of each class and output
some information to the console.
